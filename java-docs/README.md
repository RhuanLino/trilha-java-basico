# Java-Docs

Arquivo .java com exemplo e modelo de para fazer documentação de códigos.

#### Comentário em linha

```java
  //Este é um comentário em linha.
```

#### Comentário em múltiplas linhas

```java
  /*
  * Este é um exemplo
  * de comentários em
  * múltiplas linhas.
  */
```

  /**
  * Este é um exemplo
